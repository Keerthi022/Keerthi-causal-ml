# Uplift Modeling — Hillstrom Email Campaign

## Problem
Identify which customers actually respond to marketing 
campaigns — not just who would have converted anyway.

## Dataset
Hillstrom Email Marketing Dataset — 64,000 customers 
across treatment and control groups.

## Approach
Customer segmentation into 5 quintiles using:
- S-Learner
- T-Learner  
- X-Learner
- Class Transformation Method
- Neural Network implementation

## Key Insight
Uplift modeling goes beyond prediction — it identifies 
the **causal effect** of treatment on each customer.
Helps target only customers who change behavior 
because of the campaign.
